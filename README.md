# Kunal Ramdas Jadhav - Personal Portfolio Website

A premium, professional, and responsive personal portfolio website built using **React JS + Vite**. 

This portfolio showcases expertise in **Applied Machine Learning**, **Computer Vision**, **Intelligent Systems**, and **Android Development**, custom-tailored for roles in Aerospace, Defense, and elite AI startups.

---

## 🚀 Key Features

- **Live YOLOv8 Telemetry Canvas**: Interactive simulated object detection telemetry running in real-time over the hero portrait image.
- **Premium Light Engineering Theme**: Clean grids, technical blueprint elements, and vibrant blue highlights inspired by high-end engineering dashboards.
- **Responsive Layout**: Pixel-perfect scaling across desktop, laptop, tablet, and mobile viewport sizes.
- **Custom Interactive Modals**: Detail-rich case studies for high-impact projects.
- **Highly Interactive UI**: Staggered cards entry animations, micro-interactions, and a color-reveal hover effect on the hero telemetry panel.
- **100% Warning-Free**: Optimally structured layout with zero React compiler or HTML console syntax warnings.

---

## 🛠️ Tech Stack

- **Framework**: React JS (bootstrapped with Vite)
- **Styling**: Vanilla CSS (custom design system)
- **Canvas Rendering**: HTML5 Canvas API (custom interactive simulation)
- **Deployment**: Vercel

## 📦 Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kunalj2558/My-Portfilo.git
   cd My-Portfilo
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the local development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
